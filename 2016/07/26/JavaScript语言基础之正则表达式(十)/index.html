JavaScript语言基础之正则表达式(十)

Created at 2016-07-27 Updated at 2017-01-06 Category Front-End Tag JavaScript

  • Regex

    • a.[]一个字符的范围
    • b.有顺序要求的
    • c.\w==[a-zA-Z0-9_] \d==[0-9]
    • d.{count}设置匹配数量比如\w{5}，{c1,c2}

    • f.()的作用就是为了分组匹配
    • g.+代表的是1-N个，*代表的是0-N个
    • h.?代表该字符要不没有要不就有一个
